# 禁欲カウンター (Abstinence Counter)

日々の禁欲達成を匿名で記録し、同じ目標を持つ仲間との連帯感を醸成するパブリックカウンターアプリです。

**デプロイ先 URL:** [https://iqjbc-7aaaa-aaaaj-qnnsa-cai.icp0.io/](https://iqjbc-7aaaa-aaaaj-qnnsa-cai.icp0.io/)

## コンセプト
自己成長のため、あるいは願掛けのため... 様々な理由で「禁欲」に取り組む人々がいます。その日々の小さな達成を、ICP (Internet Computer) 上のパブリックカウンターに刻むことで、「自分だけじゃないんだ」という連帯感を得て、モチベーションを維持する一助となることを目指します。

## 利用ユーザー想定
*   自己規律を高めたいと考えている人
*   日々の小さな目標達成を記録したい人
*   匿名性を保ちつつ、同じ目標を持つ仲間を感じたい人

## メリット・特徴
*   **匿名参加:** Internet Identity 不要で、誰でも気軽に達成を記録できます。
*   **ガス代不要:** ICP のリバースガスモデルにより、ユーザーは無料でカウンターを増やすことができます。
*   **透明性:** カウンターの値はオンチェーンで管理され、誰でも確認できます。
*   **連帯感:** 総カウントが増えていくのを見ることで、「みんなも頑張っている」と感じられます。
*   **真の分散化:** フロントエンドもバックエンドも ICP 上で動作しています。

## ICP 活用ポイントと優位性
*   **バックエンド Canister (Motoko):** カウンター値の保存とインクリメントロジックをスマートコントラクトとしてオンチェーンで実行。
*   **フロントエンド Canister (Assets):** Web アセットを ICP 上にホスティングし、サーバーレスで Web アプリを提供。
*   **リバースガスモデル:** ユーザーにガス代負担を強いることなく、無料でのインタラクションを実現。これは多くのユーザーに参加を促す上で大きな利点です。
*   **透明性:** カウンター値がブロックチェーン上で管理されることによる信頼性。

## 使い方
1. 上記 URL にアクセスします。
2. 「現在の総達成パワー」を確認します。
3. その日に目標を達成できていれば、「達成！ (+2 ?)」ボタンをクリックします。
4. カウントが増加するのを確認します。

## 既知の問題点
*   **カウンターが +2 される:** 現在のバージョンでは、技術的な問題（フロントエンドのイベント処理または非同期通信の重複実行と推測）により、1回のクリックでカウントが意図せず2増加します。原因調査・修正が今後の課題です。

## ローカルでの実行方法
1.  `dfx` (バージョン 0.20.1 推奨), `node`, `npm` をインストールします。
2.  `git clone https://github.com/SYOGAKU-TOUSHI/your-repo-name.git` (リポジトリ名に合わせてください)
3.  `cd your-repo-name`
4.  `npm install`
5.  `dfx start --background --clean`
6.  `dfx deploy`
7.  表示されるローカル URL にアクセスします。
